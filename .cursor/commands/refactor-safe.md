# /refactor-safe — Refactor safe (локальный)

Допускается только локальный рефакторинг без изменения поведения.

Ограничения:
- Никаких переименований файлов/папок.
- Не трогай форматирование всего проекта.
- Не меняй публичные контракты (API/DTO), если не попросили.
