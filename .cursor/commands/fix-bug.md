# /fix-bug — Fix bug (минимально и безопасно)

Почини баг с минимальными изменениями.

Ограничения:
- Не делай рефакторинг «по пути».
- Сначала добавь тест, который воспроизводит баг (если возможно), потом фикси.

В конце:
- Причина бага (1–2 предложения)
- Что изменено
- Тест(ы), подтверждающие фикс
