# /add-tests — Add tests (Jest)

Добавь/обнови тесты для текущих изменений.

Ограничения:
- Не меняй прод-код, если это не нужно для тестируемости (если нужно — объясни).
- Добавь минимум:
  - unit-тест(ы) на сервис/логику
  - e2e-тест на основной happy-path
- Тесты должны быть воспроизводимыми.

В конце:
- Какие тесты добавлены
- Команды запуска
