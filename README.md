# URL Shortener + Analytics (Nest.js + Next.js) на Vercel

Этот репозиторий — учебный проект для освоения Nest.js (backend) + Next.js (frontend) с деплоем на Vercel.
База данных — Supabase Postgres (Prisma). Cron-задачи — через Vercel Cron endpoints.

## Быстрый старт
1) Открой `docs/IMPLEMENTATION_CHECKLIST.md` и иди сверху вниз.
2) В Cursor (New Chat) используй команды:
   - `/plan` → сначала план
   - `/implement-backend` → реализация
   - `/add-tests` → тесты
3) В Codex используй `AGENTS.md` + workflows (см. `docs/WORKFLOWS.md`).

## Документация
- `docs/PROJECT.md` — контекст, архитектура
- `docs/IMPLEMENTATION_CHECKLIST.md` — чек-лист прогресса
- `docs/STATUS.md` — журнал
- `docs/WORKFLOWS.md` — шпаргалка для Codex
- `AGENTS.md` — инструкции для Codex
